# California House Price Prediction

##  Project Overview
This project is an **end-to-end Machine Learning application** that predicts housing prices in California based on demographic and geographic features.

---

## Problem Statement
The goal is to predict the **median house value** using features such as income, location, and housing characteristics.

---
